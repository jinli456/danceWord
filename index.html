Usage Tips

  • 1.You need link d.css and d.js after jquery.
  • 2.Just add class as danceWord-s* in label, such as div,p,span,h1,h2...(pure text in label)
  • 3.If you are not content with default animation, you can extend it in script $.extend(danceword,{interval:200,loop:true}).
  • 4.Avoid reuse onload function in you page.
  • 5.Avoid use any label in label that contain danceWord-s*.
  • 6.DanceWord is friendly to most browser and mobile device.
